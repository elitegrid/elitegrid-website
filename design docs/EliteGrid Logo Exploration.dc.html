Four original mark concepts

Each concept is designed to be trademarkable, scalable to 16px, and distinctly tied to the EliteGrid product.

Option A Recommended
EliteGrid

Pinnacle

Three ascending columns — left to right, shortest to tallest. The glowing dot above the peak marks the "elite" summit. Reads as data columns (grid) + excellence (ascending scale). Highly scalable.

✓ Works at 16px ✓ No prior art found ✓ All 3 colorways
Option B
EliteGrid

Row Select

Three rows — the middle one selected, signified by a left-edge accent bar. It's literally a row selection UI pattern turned into a mark. Instantly communicates "data grid" and interactive selection.

✓ Product-native concept ✓ Unique silhouette
Option C
EliteGrid

The E

Three horizontal bars — long, short, long — a Bauhaus interpretation of the letter E. No vertical stroke, just the three data-row-like strokes. Monogram mark that doubles as a table visual. Works beautifully in a circle.

✓ Monogram + grid dual read ✓ Circle variant
Option D
EliteGrid

Sortmark

Two faded data rows (the grid) + one bright vertical column with an ascending sort arrow (the action). Captures the most-used grid feature in a single mark. Clean separation of data and column-level control.

✓ Feature-driven concept ⚠ Complex at 16px
