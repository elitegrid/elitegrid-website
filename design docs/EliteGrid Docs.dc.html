Installation

Set up EliteGrid in your project in under five minutes. The core package ships as pure TypeScript with zero runtime dependencies — no setup required beyond a single install command.

Prerequisites

Before you begin, make sure your environment meets these requirements:

Node.js
≥ 16.0
TypeScript
≥ 4.7 (rec.)
React / Vue
18+ / 3+
💡
TypeScript is strongly recommended. EliteGrid's grouped config API is designed to be used with TypeScript — you'll get full inference on every option, event, and plugin.

Install the Package

Choose the adapter for your framework. The @elitegrid/core package is always installed automatically as a peer dependency.

{{ pmTabsEl }}

      


      

        📦
        
Zero dependencies. @elitegrid/core has no runtime dependencies. The adapter packages only add the framework bindings you're already using.
